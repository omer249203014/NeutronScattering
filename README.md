# Neutron Scattering Energy Calculator

This program computes the final energy of a neutron after an elastic collision with a target nucleus. 
The user provides the target mass number (A) and the scattering angle (theta, in degrees), 
and the program calculates and displays the neutronâ€™s final energy in MeV. The initial neutron energy is assumed to be 1 MeV.

## How to run
1. Make sure you have Python 3 installed.
2. Run `neutron_scattering.py`.
3. Enter the target mass number (A) and the scattering angle (theta) when prompted.
4. The program will print the neutron's final energy in MeV.
